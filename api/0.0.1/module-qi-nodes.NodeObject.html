Class: NodeObject

qi-nodes. NodeObject

new NodeObject() → {module:qi-nodes.NodeObject}

These objects, or types of them, are used in composite patterns. Each instance of module:qi-nodes.NodeObject is aware of it's own position within a tree of nodes. If any tree has a root, that root is propogated to all leaves of that tree. When a leaf is removed from a rooted tree, the previous root reference is removed for all nodes contained in that leaf.
